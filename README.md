# Abstract :

The task of land use and land cover(LULC)
classification presents a significant challenge because it
requires models that are both accurate and computation-
ally efficient. In this paper, we introduce FractalCLS, a
compact convolutional-attention model that is inspired by
fractals. This model utilizes hierarchical fractal blocks
that apply windowed self-attention at multiple smaller
scales. With only 6.21 million parameters, FractalCLS
achieves 96.70% accuracy on the EuroSAT test dataset.
This demonstrates a balance between performance and
model size. To further verify these results, we perform
qualitative auditing by utilizing Grad-CAM. The results
show that these fractal-inspired designs can be an alter-
native to traditional CNN or transformer-based models
for developing high-performance, efficient LULC clas-
sification models for resource-constrained environments.
The code for the paper is available at GitHub.
